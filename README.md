### 👨‍💻 About Me

- 🌍 Based in **Nairobi, Kenya** — building for global teams
- 🚀 Currently shipping with **Next.js 15 · TypeScript · Supabase · TailwindCSS**
- 💼 Open to **freelance, contract & remote full-time** opportunities
- 🛠️ Recent work I'm proud of:
  - 🏨 **TricomHub** — hotel booking platform with M-Pesa & Stripe
  - ☕ **ByteSpaceCyber** — full-stack cyber café SaaS for Nairobi
  - 🗄️ **MuriukiDB** — browser-based RDBMS in TypeScript (SQL lexer, parser, B-Tree indexing, REPL)
- 💬 Ask me about **React, Next.js, Supabase, M-Pesa integration, or UI/UX**
- 🧠 Currently going deeper on **MERN** & multi-agent AI systems
- ⚡ Fun fact: I built a relational database engine from scratch — for fun
